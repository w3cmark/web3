# Web3相关资料


## 前言

2022年，互连时下热议当属Web3.0——新一代互联网的简称。人们对新一代互联网的定义：不仅能够组合信息，同时也能像人类一样读懂信息，并以类似人类的方式自主学习于知识推理，从而为人类提供准确可靠的信息，从而使得人与互联网的交互更加自动化、智能化及人性化。而这样的场景设想也诞生了包括元宇宙、crypto、NFT、DAO 等新兴概念。

本文旨在收集这些相关概念的资料，供自己以及感兴趣的人学习。
